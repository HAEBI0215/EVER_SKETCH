# EVER-SKETCH


헤비님의 팬게임 제작 리포지토리입니다.<br>
This is hebi.’s fan game development repository.

이 게임은 기본적으로 hebi.님의 Chroma 앨범의 서사와 전하고자 하는 뜻을 담았습니다.<br>
This game is fundamentally based on the narrative and message of hebi.’s Chroma album.

이 게임은 비영리 게임입니다.<br>
This is a non-commercial game.

이 게임의 무단 복제, 허가없는 유출 및 배포는 금지합니다.<br>
Unauthorized copying, distribution, or leakage of this game is strictly prohibited.
